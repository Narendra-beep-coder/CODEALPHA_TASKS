# CODEALPHA_TASKS

This repository contains tasks completed as part of the **CODE ALPHA** internship, focused on Python-based cybersecurity and networking projects.

## Overview

The main branch groups multiple internship tasks in a single place so they can be tracked, updated, and showcased together.  
Each task lives in its own folder with the relevant Python files and, where needed, its own README and resources.

## Tech stack

- **Language**: Python (100% of the repository).
- **Use cases**: Learning and practicing network traffic handling, security tooling, and secure coding patterns in Python.
